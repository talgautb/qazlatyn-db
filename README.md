# qazlatyn-db

[![NPM version][npm-image]][npm-url]

> Raw data of kazakh latin alphabet (qazaq latyn)

## Install

    npm install --save qazlatyn-db

## Usage

```js
import qazLatyn from 'qazlatyn-db';

qazLatyn['ә']; // á
qazLatyn['ң']; // ń
qazLatyn['қ']; // q

```
